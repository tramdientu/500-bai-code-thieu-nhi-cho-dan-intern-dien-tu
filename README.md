# 🚀 ESP32 Learning Projects

Tổng hợp các bài học và project thực hành với ESP32 sử dụng PlatformIO.
Phù hợp cho người mới bắt đầu đến nâng cao.

---

## 📚 Danh sách bài học

### 1. 🕒 ESP32 Clock

* Hiển thị thời gian thực
* Đồng bộ NTP (nếu có)

---

### 2. 🌡️ ESP32 DHT22

* Đọc nhiệt độ & độ ẩm từ cảm biến DHT22

---

### 3. 💡 ESP32 DIMMER LED

* Điều chỉnh độ sáng LED bằng PWM

---

### 4. ⚙️ ESP32 PWM Servo

* Điều khiển servo bằng PWM

---

### 5. 🔢 ESP32 Clock 7 Segment

* Hiển thị thời gian bằng LED 7 đoạn

---

### 6. 🔄 ESP32 OTA

* Update firmware qua WiFi (Over-The-Air)

---

### 7. 💾 ESP32 Flash Storage

* Lưu dữ liệu vào bộ nhớ flash

---

### 8. 🧱 ESP32 LED Matrix

* Điều khiển LED ma trận

---

### 9. 🔊 ESP32 DFPlayer

* Phát âm thanh với module DFPlayer

---

### 10. 📡 ESP32 S3 + R8N16

* Làm việc với board ESP32-S3

---

### 11. 🎮 ESP32 MPU6050 TFT

* Đọc cảm biến MPU6050
* Hiển thị lên màn hình TFT

---

### 12. 🖥️ ESP32 TFT LCD Adjust

* Tùy chỉnh hiển thị LCD

---

### 13. 📶 ESP32 ESP-NOW

* Giao tiếp không dây ESP-NOW

---

### 14. 🔗 ESP32 Bluetooth HID

* Giả lập thiết bị HID (bàn phím/chuột)

---

### 15. 🖱️ ESP32 BLE Mouse

* Điều khiển chuột qua Bluetooth BLE

---

## ⚙️ Cách sử dụng

```bash
# Build project
pio run

# Upload firmware
pio run -t upload
```
